Real KITTI Data Layout

After you provide KITTI data, try paths like these: 

user_data/image_2/000000.png
user_data/velodyne/000000.bin

The viewer reads KITTI Velodyne .bin files as repeated float32 values: x forward, y left, z up, reflectance.
